# Check Signing Status

## Usage examples
- "What's the status of the NDA I sent to jane?"
- "Has the MSA been signed yet?"
- "Show me all pending envelopes"
- "List everything I sent this week"

## What you'll see
- Envelope status: draft, sent, partially signed, completed, voided, expired
- Per-signer status: who has signed, who hasn't
- Timestamps for each action
